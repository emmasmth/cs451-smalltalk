## To use Smalltalk interactive mode:

  gst -q
  'Hello World!' printNl !

  to exit interactive mode, hit ctrl-d

## To execute smalltalk program (.st file extension)

  gst file.st

## Comments in Smalltalk are enclosed in double quotations 

  "this is a comment"
